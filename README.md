## Emotion Detection from Text

Models implemented:
- TF-IDF + Logistic Regression
- LSTM (from scratch)
- DistilBERT (fine-tuned)

Best Accuracy:
- DistilBERT: 92.55%

Deployment:
- Streamlit web app for real-time emotion detection
